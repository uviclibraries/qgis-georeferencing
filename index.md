# Georeferencing with *QGIS*

- Pre-workshop activities: 5 min 
- Introductory presentation: 20-30 min
- Hands-on activities: 60 min

## Why QGIS? 

[QGIS](https://qgis.org/download/){:target="_blank"} is a free, open source desktop Geographic Information Systems (GIS) package that allows users to analyze and edit spatial information and create and export maps.

## This Workshop
**It is highly recommended to:**
- **Use a second monitor or print out the workshop activities**
- **Use mouse with a scroll wheel**

This introductory workshop will cover the basics of georeferencing in QGIS, including understanding the uses and possibilities of georeferencing, using a georeferencing tool to georeferenced a historical map, and using an existing open data source as target data to georeference an image.

### Note
With *QGIS* someone could:
- create beautiful maps
and/or
- conduct spatial analysis

*QGIS* has hundreds of tools and possiblites...
This workshop is an **Introduction** to *QGIS* with georeferencing

## Learning objectives
- Define the basics of a Geographic Information System and raster data
- Identify and navigate key interface elements of QGIS Georeferencer tool
- Define key terms and processes of georeferencing (Target data, Ground Control Point, Transformation algorithm)


## Learning Outcomes
Using *QGIS*, participants will:
- Use QGIS Georeferencer tool to georeference a historical map of Victoria, BC
- Use an existing open data source as target data to georeference
